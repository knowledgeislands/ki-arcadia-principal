# Isometric Tiles

## Overview

The Isometric Tiles are a modular asset set for constructing map-like visual representations of Knowledge Islands territories,
architectures, and systems. Tiles are designed on a consistent isometric grid so that they compose cleanly into larger scenes - individual
islands, full archipelagos, or abstract system maps.

The set covers terrain, structures, connectors, and props at multiple scales, and includes guidance on seamless edges, water alignment,
height layers, and grid composition.

---

## Basic Tile Set

The basic set covers the foundational terrain and building types:

**Terrain:** basic land, elevation and cliffs, coastal tiles, ocean tiles, special terrain (waterfalls, volcanic, etc.)

**Buildings:** small structures (1x1), medium structures (1x2 / 2x1), large structures (2x2 / 3x2), landmarks (4x4)

**Tile sizes:** 1x1, 1x2, 2x1, 2x2, 2x3, 4x4

**Colour palettes:** Archipelago, Savanna, Natural Colours, Glass and Stone

## Structures and Terrain Set

An expanded set with greater terrain variety and additional structure types:

**Terrain:** grassland, forest, hills and rocks, coastal, cliffs, sand and dry, special

**Structures:** small (1x1), medium (1x2 / 2x1), large (2x2 / 3x2), landmarks (4x4)

**Utility and connectors:** paths, bridges, docks, walls

**Details and props:** trees, rocks, barrels, crystals, pillars, gates

**Large compositions:** example multi-tile arrangements for reference
