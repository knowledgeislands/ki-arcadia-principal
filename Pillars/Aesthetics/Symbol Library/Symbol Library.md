# Symbol Library

## Overview

The Symbol Library is the canonical reference for the visual symbols used across Knowledge Islands outputs. Each symbol carries a defined
conceptual meaning drawn from the geographic metaphor at the heart of the model - and must be used consistently so that viewers build
reliable associations across diagrams, interfaces, presentations, and communications.

The library provides two layers of asset: a reference sheet mapping symbols to their meanings, and a set of standalone illustrations for use
in documents and interfaces.

For the full symbolic semantics table and terrain language, see [[Visual Style/Visual Style|Visual Style]].
