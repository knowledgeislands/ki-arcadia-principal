# Governance

## Overview

Arcadia governs itself using the **council model** as defined in [[Concept]]. The council is Arcadia's sole ratification authority - all changes to canonical knowledge pass through it via the Enactment Process.

---

## Council

| Attribute           | Value                                                                                   |
| ------------------- | --------------------------------------------------------------------------------------- |
| Model               | Council - see [[Concept]]                    |
| Operating mechanism | [[Knowledge Capital/Governance/Processes/Enactment Process]] |
| Membership          | Open to principals of any archipelago that contributes to or builds on Arcadia          |

**Current membership (April 2026):**

| Member | Role            | Archipelago  |
| ------ | --------------- | ------------ |
| Kit    | Founding member | Arcadia, Kit |

## Activities

The [[Knowledge Capital/Governance/Activities/Activities]] folder holds Arcadia's timing configuration for automated activities and the adoption stubs for non-adopted groups. The Schedule note defines the day-type taxonomy that automations read from daily note frontmatter. Veto stubs for Email and Linear sit in their respective subfolders.

---

## Agents

The [[Knowledge Capital/Governance/Agents/Agents]] folder holds the island-specific material that shapes how AI agents interact with Arcadia. Communication Style gives Claude a reference for Kit's voice; Canonical Meta Notes provides the ordered reading list that Knowledge Rebuild uses to ground an agent when starting cold.

---

## Conventions

The [[Knowledge Capital/Governance/Conventions/Conventions]] folder holds the island-specific vocabulary and routing rules that supplement the generic KI conventions. The Glossary decodes Knowledge Islands terminology; Routing Rules records the filing decisions that recur in Arcadia's specific domain structure.

---

## Processes

The [[Knowledge Capital/Governance/Processes/Processes]] folder holds Arcadia's instance of the KI process definitions. Currently it contains the [[Knowledge Capital/Governance/Processes/Enactment Process]] - Arcadia's ratification mechanism for changes to canonical knowledge.

---

## Tools

The [[Knowledge Capital/Governance/Tools/Tools]] folder holds the concrete infrastructure configuration for this island. Integrations lists the external services connected and their MCP identifiers; Physical Locations records the three filesystem paths - text store, binary store, and Cowork working folder - that make up Arcadia's physical footprint.
