# Pillars-Resources Boundary

## Overview

The boundary between Pillars and Resources is the most important routing decision in the island. This note defines the rules specific to Arcadia.

---

## Rules

- `Pillars` = knowledge that Arcadia owns and produces — the Knowledge Islands concept, governance model, architecture, and methodology; Arcadia's own configuration and identity
- `Resources` = external reference that exists independently of Arcadia — people records, external tools, reference material from outside the KI ecosystem
- Links between Pillars and Resources are bidirectional where relevant
- If a Resources note accumulates internal knowledge, that content belongs in a Pillars note that references it
