# Reading Order

If you are reading for the first time, follow the sequence below — each step establishes the vocabulary and model that the next one builds on. Once familiar with the content, the wikilinks within each note are the navigational paths and any note can serve as an entry point.

  - [[Format]]
  - [[Types]]

- [[Structure]] — the island's geography: Library, Harbour, and Streams
