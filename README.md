# Arcadia Principal

Arcadia Principal is the canonical knowledge base for Knowledge Islands. It develops, proves, and records the philosophy and model in a living island; it does not mandate a particular tooling implementation.

## Place in the Knowledge Islands ecosystem

Arcadia Principal is the authoritative source for the Knowledge Islands philosophy and model. Patterns that need a reusable technical realisation can be generalised by the [KI Agentic Harness](https://github.com/knowledgeislands/ki-agentic-harness), while proven concepts and implementation evidence can be formalised as normative contracts by [KI Specifications](https://github.com/knowledgeislands/ki-specifications).

The [KI Website](https://github.com/knowledgeislands/ki-website) is an autonomous publication of selected material from all three source repositories; publication never transfers canonical ownership. [GDR-KI-ARCADIA-002](Admin/Governance/Decisions/GDR-KI-ARCADIA-002-knowledge-islands-ecosystem-fundamentals.md) defines the shared boundary and publishing flows mirrored across the four repositories.
