# Calendar

## Overview

The Calendar folder contains all time-stamped notes: daily notes, meeting notes, session digests, weekly reviews, and monthly summaries.

**Daily notes** are the primary anchor for the day - linking out to all other Calendar notes for that date. Path: `Calendar/YYYY/YYYY-MM MonthName/YYYY-MM-DD DayName.md`

**Meeting notes** are filed alongside daily notes in the same month folder - one note per meeting. The daily note references each meeting note by wikilink; it does not duplicate meeting content. Path: `Calendar/YYYY/YYYY-MM MonthName/YYYY-MM-DD Meeting Name.md`

**Session digests** capture substantive AI-assisted work sessions - one note per session, filed in the same month folder and referenced from the daily note by wikilink. Path: `Calendar/YYYY/YYYY-MM MonthName/YYYY-MM-DD Session - Topic.md`

**Weekly notes** are filed in the month folder where that week's Monday falls. Path: `Calendar/YYYY/YYYY-MM MonthName/YYYY WXX.md`
