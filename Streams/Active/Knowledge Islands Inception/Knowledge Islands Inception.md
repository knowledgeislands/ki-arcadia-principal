## Design Decisions

| Decision | Resolution |
| --- | --- |
| Skill name for Arcadia | `arcadia` |
| Cascade model | Copy model — each principal holds a local copy of Arcadia's governance as its baseline; principals pull updates, Arcadia does not push |
| `Pillars/Knowledge Islands/` scope | Conceptual framework only — no Arcadia-specific content; generic and portable across all islands |
| `Pillars/Knowledge Capital/` | Universal folder name for every island's self-knowledge — the island's realisation of the KI framework; replaces the old `KB Specifics` concept |
| `Known Lands` | Universal name for the island map across all KBs — Arcadia's objective record and each principal's personal navigator's chart share the same name |
| KM location | `Pillars/Knowledge Islands/Governance/` — KM methodology is part of the KI framework, not a Productivity sub-topic; renamed from `Knowledge Management` to `Management` under its new parent |
| Fate of `Pillars/Productivity/` after KM moves | Deferred — retains `Working Practices/`; no structural change for now |
