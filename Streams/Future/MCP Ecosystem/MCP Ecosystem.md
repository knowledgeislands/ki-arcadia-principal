# MCP Ecosystem

## Overview

A survey stream to identify MCP servers and registries relevant to the island's connected tools. The MCP ecosystem is growing quickly; a periodic survey ensures the island's tooling stays current and any high-value connectors are identified before they are needed.

---

## Governance

This stream follows the [[Knowledge Islands/Processes/Enactment Process|Enactment Process]].

---

## Checklist

- [ ] **Registry survey** — Review the main MCP registries for connectors relevant to the island's current and planned tools: [mcpservers.org](https://mcpservers.org/), [mcplane.com](https://mcplane.com/), [MCP Dock](https://apps.apple.com/us/app/mcp-dock/id6748305262?mt=12), [FastMCP](https://gofastmcp.com/getting-started/welcome)
- [ ] **Gap analysis** — Cross-reference survey findings against the connected tools in the Charter; identify any high-value connectors not yet adopted
- [ ] **Adoption candidates** — For any strong candidates, create an investigation item in the relevant tools stream or propose directly via the Enactment Process
