# External Integrations

## Overview

A holding stream for external tools and MCP connections worth evaluating. Items here are at the investigation stage — nothing has been trialled yet. Each warrants a session to assess fit before any adoption decision.

---

## Governance

This stream follows the [[Pillars/Knowledge Islands/Governance/Processes/Enactment Process|Enactment Process]].

---

## Intended Destinations

- [ ] **n8n** — Investigate [n8n](https://n8n.io/) as a workflow automation layer; assess whether it adds value over the existing Cowork scheduled task approach or solves problems the current stack cannot
- [ ] **Hex** — Investigate [Hex](https://hex.tech/) for data analysis and notebook-style outputs; assess relevance to island workflows
- [ ] **MCP ecosystem survey** — Survey the MCP registries for integrations relevant to the island's connected tools: [mcpservers.org](https://mcpservers.org/), [mcplane.com](https://mcplane.com/), [MCP Dock](https://apps.apple.com/us/app/mcp-dock/id6748305262?mt=12), [FastMCP](https://gofastmcp.com/getting-started/welcome)
