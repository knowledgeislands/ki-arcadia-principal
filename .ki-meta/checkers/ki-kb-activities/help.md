# ki-kb-activities

Author, audit, and manage Activity notes in a Knowledge Islands base — the operational record of what automation, scheduling, and agentic work a base has adopted.

**Invoke:** `ki-kb-activities audit | conform | help | educate | new <name> | refresh`

**Modes:**

- `AUDIT`
- `CONFORM`
- `EDUCATE`
- `HELP` — explain this skill and stop; the default when no mode is given (then routes, if interactive)
- `NEW`
- `REFRESH`

**See also:** For the KB zone structure use `ki-kb`; for skill authoring use `ki-skills`; for the harness bundle layout use `ki-harness`.
