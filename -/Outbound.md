# Outbound

## Overview

`-` is the outbound staging area of the canonical Knowledge Islands model - the counterpart to the inbound `+` - where produced artefacts leave the island: session digests, compiled outputs, and material prepared for use elsewhere. Like `+`, it is staging rather than a zone, so it is exempt from the same-name index rule; this note documents it minimally so the island carries the full structure.

The zone is introduced empty for now. Session digests currently remain sibling Calendar notes (see [[CLAUDE]]); when digest output migrates here it lands under `-/_DIGESTS/`, alongside the move of governance into [[Admin]].
